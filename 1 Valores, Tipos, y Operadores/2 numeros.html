Números

Los valores de tipo número son el 13, 15 o 10, como se puede ver el valor es un número, para guardar un solo número javascript utiliza 64 bits, eso quiere decir que para un solo número hay 64 espacios vacios los caules pueden contener una serie de 1 y 0 que van a representar un número

ejemplo

Espacios en memoria Uno Dos Tres Cuatro Cinco Seis Siete Ocho Nueve Diez
bits 1 2 4 8 16 32 64 128 256 512

si se quiere representar el número 75 en bits usando la anterior tabla

Espacios en memoria Uno Dos Tres Cuatro Cinco Seis Siete Ocho Nueve Diez
bits 1 2 4 8 16 32 64 128 256 512
75 en bits 1 1 0 1 0 0 1 0 0 0

Podemos ver que cada espacio representa un número entero que cada vez se va hacen mas grande y que cada número se multiplica por 2 lo cual genera el número siguiente (1x2 =2, 2x2=4, 4x2=8, 8x2=16), entonces si se suman los bits que tienen el número 1 los cuales representan el número 75, pasa lo siguiente:

1 + 2 + 8 + 64 = 75

Entonces podemos concluir que sí utilizamos la tabla anterior para representar valores numéricos solo podemos hacer 2^10 (2 elevado a 10) combinaciones, lo cual 1024 sería el número mayor que podemos representar como un solo valor numérico y -1023 el número menor que podemos representar como un solo valor numérico.

Una computadora puede representar 2^64 (2 elevado a 64) combinaciones, lo que nos dejaria un número enorme que podemos representar como un solo valor numérico

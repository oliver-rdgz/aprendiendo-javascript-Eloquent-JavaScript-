Valores Strings

Como se dijo anteriormente cuando se hace una compra en linea se hace hay diferentes categorias como juguetes, electrodomesticos, deportes entre otros, así podemos encontrar diferentes tipos de valores en javascript, el string son aquellos valores que representan texto. En javascript se representan los textos de las siguiente forma:

  • 'Hola soy un texto simplon'
  • "Soy el segundo texto de preuba con doble sabor"
  • `soy un texto ¿ y que? ademas soy invertido`

Si notamos que los valores de tipo textos estan encerrados en comillas las cuale son:

  • '' comillas simples
  • "" comillas dobles
  • `` comillas invertidas

como el texto está encerrado entre comillas surgió algo llamado secuencia de espace en javascript lo que permite escapar del encierro de las comillas y que su comportamiento sea diferente a un simple texto, esta secuencia de escape se realiza con la barra invertida (\).

Ejemplo de secuencia de escape

Como se ingresa el texto a javascript: "Hola soy la prueba \npara la secuencia de escape"

Como lo muestra javascript: "Hola soy la prueba
para la secuencia de escape"

cuando se colaca dentro de un texto \n lo que hace es un salto de linea, entonces lo que hizo \n fue:
linea 1: Hola soy la prueba \n(saltar linea)
linea 2: para la secuencia de escape

Existen muchas secuencias de escape de las cuales puedes investigar y aprender.

Unicode

Unicode es lo que utiliza javascript como sistema de codificación el cual permite convertir un simbolo que nosotros los humanos entendemos a una serie de bits que la computadora entienda y tambien de manera viceversa permite convertir los bits en simbolos.

Valor numérico

En javascript puedes tener un número de la siguiente forma:

  • Entero positivo: 12
  • Entero negativo: -5
  • Decimal: 2.5

Todos los valores anteriores son valores de tipo numérico, pero como en el caso de los juguetes que puedes tener un carro, una consola de videojuegos en los números puedes tener enteros positivos, enteros negativos y decimales todos estos valores para javascript son de tipo numérico.

¿Que se hacen con los números?

te pregunto ¿tu que puedes hacer con los números? para javascript lo mas comun y segun su opinion lo que puede hacer con los números es realizar operaciones aritmeticas, para realizar las operaciones se usan los siguientes operadores:

  • suma +
  • resta -
  • multiplicación *
  • división /

4-1+2*3/6

¿Cual es el orden que se debe realizar operación anterior?

Respuesta:

  • primero se hace la multiplicación 2*3 = 6 que daria de la siguiente forma la operación 4-1+6/6
  • segundo se hace la división 6/6 = 1 que daria de la siguiente forma la operación 4-1+1
  • Tercero se hace la resta 4-1 = 3 que daria de la siguiente forma la operación 3+1
  • Cuarto se hace la suma 3+1 = 4 que finalmente es el resultado de la operación

javascript es uno de los lenguajes de programación que realiza las operaciones aritmeticas de izquierda a derecha, pero tiene realiza primero unas operaciones que otras, si tenemos 4-2+1*5/5 lo primero que va hacer javascript es hacer la multiplicación ya que los operadores tienen en javascript una jerarquia, ya que para javascript y su pensamiento capitalista el operador * y / tienen mayor importancia que unos simples operadores de + y - cuando javascript realiza las operaciones siempre de izquierda a derecha, como se vio en la operación anterior 4-2+1*5/5 primero se hace la multiplicación antes que la división porque está más a la izquierda y la división está más a la derecha, realizamos la multiplicación 1*5 = 5 y la operación quedaría 4-2+5/5 luego se hace la división 5/5 = 1 y la operación quedaría 4-2+1 entonces como con la multiplicación y la división con la suma y la resta debemos hacer los que estan mas a la izquierda y luego hacer los que estan a la derecha, con lo anterior dicho se realiza la resta 4-2 = 2 y la operación quedaría 2+1 finalmante se hace la suma 2+1=3.

si quieres darle mayor importancia a cualquier operación solo debes ponerlos en parentesis ya que el parentesis para javascript es lo mas importante de todo más que la multiplicación y la división (4-2)+1*5/5 en esta operación lo primero que se hace es la resta 4-2 = 2 y la operación quedaría 2+1*5/5 luego se hace la multiplicación 1*5 = 5 y la operación quedaría 2+5/5 luego se hace la división 5/5 = 1 la operación quedaría 2+1 finalmente se hace la suma 2+1 = 3.

en javascripte tambien existen otros operadores, pero los anteriores son los mas utilizados

¿Qué es JavaScript?

Javascript nació en 1995 gracias al veterano y ya enterrado navegador Netscape Navigator, el cual utilizaba Javascript para agregar programas a las paginas webs.

Javascript con el tiempo ha evolucionado y es utilizado por todos los navegadores web, para que las paginas webs sean dinamicas y que el usuario no tenga que estar recargando la pagina web cada vez que haga una acción.

Ummmm otra cosa más ECMAScript y javascript basicamente es utilizado para referirse a lo mismo, pues ECMAScript es un estandar para decir como se debe utilizar javascript y cuales son sus reglas, pero igualmente javascript sino sigue el estandar ECMAScript no va a funcionar, así que vendria siendo lo mismo decir ECMAScript y javascript.

También Javascript a evolucionado a los servidores con nodejs, el cual permite utilizar javascript fuera del navegador solo con tener instalado nodejs.
